A string that describes the type of content being sent, such as:

  • TYPE_COMMENT

  • TYPE_FORUM_POST

  • TYPE_REPLY

  • TYPE_BLOG_POST

  • TYPE_CONTACT_FORM

  • TYPE_SIGNUP

  • TYPE_MESSAGE

  • TYPE_PINGBACK

  • TYPE_TRACKBACK

  • TYPE_TWEET

You may send a value not listed above if none of them accurately describe your content.
